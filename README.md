# ap-ui
ap-ui vue-components.

## Installation
```
npm install ap-ui
```

## Import
```
import 'ap-ui/dist/ap-ui.min.css';
import { Loading, SolidArrow... } from 'ap-ui';
```

## Demo
下载该项目，在文件根目录下依次输入命令行：
```
npm install
npm run dev
```
然后在浏览器里输入`http://localhost:8089`，即可看到组件 demo。
